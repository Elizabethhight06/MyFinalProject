Hello :) This program is intended to allow you to search through Bible Verses from the KJV Version. Please take into account that the spelling is
different in the KJV version. The language of this program is intended to be simple and consistent. A menu is presented to prompt the user fir specific
funtctions. These functions do some things with the verses. These functions include: a split record, search for keyword(or phrase), search for verse, 
search for book, random verse, random verse in book, and bible statistics (longest book, shortest book, average verse length, longest verse, shortest verse.)
